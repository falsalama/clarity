## Description

Clarity is a quiet practice app for daily reflection, meditation, learning, and Buddhist-inspired mind training.

Use it to keep a daily practice rhythm, listen to meditation sounds and teachings, reflect on what is moving through the mind, and return to a clearer, more compassionate way of meeting experience.

Core features are free:
- Daily practice
- Reflection capture
- Meditation timer
- Calendar and observances
- Learning, wisdom, and compassion entries
- Audio and practice support
- Portrait and capsule features

Clarity Reflect is optional paid access for deeper Cloud Tap responses. It uses selected redacted text to generate practice-focused reflection, perspective, options, questions, and talk-it-through support. Audio and raw transcripts stay on your iPhone. Only the selected redacted text is sent when you choose a Cloud Tap response.

Clarity is not medical advice, therapy, or crisis support. It is a practice companion for reflection, mindfulness, and Buddhist learning.

## App Review Notes

Clarity includes optional in-app purchases for Clarity Reflect.

Core app features remain available without purchase. Paid access unlocks Cloud Tap reflection tools: practice-focused perspective, Options and Questions, Talk it through, deeper reflection responses, and expanded practice audio access.

Cloud Tap sends only selected redacted text when the user chooses a Cloud Tap response. Audio and raw transcripts remain on device.

The app uses HealthKit only for user-authorized, on-device mindfulness and health pattern support. Health data is not sold or used for advertising.

The app creates an anonymous Supabase account for Cloud Tap access and entitlement checks. Users can delete the current cloud account from Profile > Privacy > Delete Cloud Account and Data.

The app is not medical advice, therapy, diagnosis, treatment, or crisis support.
